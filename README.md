# Advertisement
## Описание 
[Тестовое задание](https://github.com/avito-tech/adv-backend-trainee-assignment), найденное на просторах интернета и сделанное для саморазвития.

Advertisement это REST API сервис для хранения и подачи объявлений.
## Запуск

- Скопировать файл оружения `cp config.env config.env.local`
- Изменить данные доступа к БД при необходимости в файле `config.env.local`
- Запустить docker-compose `docker-compose up -d`
